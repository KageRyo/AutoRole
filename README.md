# Discord_AutoRole-Py
[DiscordBot] by Python | KageRyo Dev

AutoRole Bot - 在有人加入 DC 群的時候自動給予身分組。

### 套件：
```
$pip install discord
```

### 主程式，執行的時候請跑這個

AutoRole.py
> 主程式

### 配置檔，在這裡設定TOKEN和RoleName or RoleID吧

config.py

*TOKEN*
> 機器人的TOKEN

*RoleName or RoleID (choose one)*

> 身分組設定（名稱）
> 身分組設定（ID）
